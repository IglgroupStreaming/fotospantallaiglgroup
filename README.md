# 🖼️ Sistema de Pantalla Gigante

Sistema completo para gestionar y proyectar fotos en una pantalla gigante, con panel de administración y almacenamiento local.

## ✨ Características

- 📸 **Subida de fotos** por usuarios con drag & drop
- 🎮 **Panel de administración** para moderar contenido
- 🖥️ **Pantalla gigante** con actualización automática
- 💾 **Almacenamiento local** - no necesita servidor
- 🔄 **Actualización en tiempo real** entre pestañas
- 📱 **Diseño responsive** para todos los dispositivos

## 🚀 Cómo usar

1. Clona o descarga el repositorio
2. Abre `index.html` para subir fotos
3. Abre `admin.html` para moderar (modo administrador)
4. Abre `pantalla.html` para proyectar en pantalla gigante

## 📁 Estructura de archivos

- `index.html` - Interfaz para usuarios (subir fotos)
- `admin.html` - Panel de control del moderador
- `pantalla.html` - Pantalla de proyección
- `storage.js` - Módulo de almacenamiento local

## 🛠️ Tecnologías

- HTML5
- CSS3 (Grid, Flexbox, Animaciones)
- JavaScript (ES6+)
- LocalStorage API
